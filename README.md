# 加解密工具简单原理说明
此加解密工具采用类比特币勒索病毒基础工作原理制作，采用RSA非对称加密算法，公钥加密，私钥解密。
![](_v_images/20200415185159195_404992704.png)

# 功能说明
* 支持跨平台（MacOS、Windows、Deepin Linux v20）。
* 采用RSA非对称加密算法。
* 只能够对单个文件进行加密，暂不支持多文件批量加密。
* 支持加密各种类型的文件，包括（.doc,.docx,.txt,.png,.jpg ...）。
